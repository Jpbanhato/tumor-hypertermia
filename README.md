# tumor-hypertermia
Este trabalho tem como objetivo analisar a **biotransferência de calor no tecido tumoral** e avaliar a eficiência do processo de necrose induzido pelo aumento da temperatura. A implementação foi realizada na linguagem **C++**, apresentando um desempenho aceitável mesmo em execução serial para alguns valores de **\( h_t = 0.1 \)**.
